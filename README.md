# PulseWire

PulseWire is a technology news aggregator built with Node.js, Express, and vanilla JavaScript. It collects stories from multiple tech publications, combines overlapping coverage, and presents them in a clean single-page interface.

The app includes a lightweight backend for fetching and filtering RSS feeds, along with a responsive frontend for browsing, searching, saving, and refreshing stories. It can run locally or be deployed on Vercel for public access.
